 # mars3d-plugin-space

基于[Mars3D](http://cesium.marsgis.cn)的雷达及卫星相关可视化功能插件


说明：
  （1）satellite.js来源于https://github.com/shashwatak/satellite-js，提供SGP4 / SDP4计算所需的功能，广州欧科略有扩展修改；
  （2）mars3d-space.js 为卫星视锥体、雷达区域等cesium扩展矢量对象，广州欧科开发。


## 相关依赖
核心依赖库：Cesium、okay3d
